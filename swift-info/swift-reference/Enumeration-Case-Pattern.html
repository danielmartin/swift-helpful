9.5 Enumeration Case Pattern

An enumeration case pattern matches a case of an existing enumeration type. Enumeration case patterns appear in ‘switch’ statement case labels and in the case conditions of ‘if’, ‘while’, ‘guard’, and ‘for’-‘in’ statements.

If the enumeration case you’re trying to match has any associated values, the corresponding enumeration case pattern must specify a tuple pattern that contains one element for each associated value. For an example that uses a ‘switch’ statement to match enumeration cases containing associated values, see (swift)Associated Values.
